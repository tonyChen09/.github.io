Wireframes Halaman Utama KshowOnline

Masalah yang ingin dipecahkan melalui tampilan visual

Halaman Utama KshowOnline terdapat menu shows list, latest show, top viewed dan top rated serta terdapat 3 column menjabar kebawah dengan kategori berbeda-beda dan juga terdapat search box. Berikut penjelasannya :

  1. Show list

    2. menampilkan semua program variety dari a-z

  2. latest show

    3. menampilkan episode terbaru dari program variety

  3. top viewed

    4. menampilan acara variety dengan jumlah view terbanyak

  4. top rated

    5. menampilkan acara variety show dengan jumlah rate terbanyak

  5. search box

    6. untuk mempermudah user melakukan pencarian


Dan 3 kategori yang berbeda itu adalah

  1. Top show today

    2. menampilkan acara variety berdasarkan jumlah view dan rate terbanyak di hari itu juga

  2. Latest show

    3. menampilkan episode terbaru dihalaman utama sehingga mempermudah user dalam melakukan pencarian

  3. Random show

    4. menampilkan random show dari variety program sehingga apabila user bingung dalam menentukan variety yang mau di tonton, pilihan ini bisa berguna
